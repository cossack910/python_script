# 問題: 回文チェック (Palindrome Checker)

文字列を入力として受け取り、その文字列が**回文（前から読んでも後ろから読んでも同じ文字列）**かどうかを判定するプログラムを作成してください。

## 要件:

1. ユーザーに文字列を入力させる。
1. 入力文字列の前後のスペースや大文字小文字の違いは無視して判定する。
1. 例: "Level" や " A man a plan a canal Panama " は回文とみなす。
1. 回文であれば「回文です」、そうでなければ「回文ではありません」と出力する。

### サンプル出力

入力 1

```
文字列を入力してください: racecar
回文です
```

入力 2

```
文字列を入力してください: hello
回文ではありません
```

入力 3

```
文字列を入力してください: A man a plan a canal Panama
回文です
```
